INSTRUÇÕES - APP WEB (Streamlit)

1. Requisitos:
   - Python 3.x instalado
   - Instalar bibliotecas necessárias (uma vez apenas):

        pip install streamlit pandas openpyxl

2. Como usar:
   - Abre um terminal ou prompt de comando
   - Vai até à pasta onde estão os ficheiros
   - Corre o seguinte comando:

        streamlit run app_contagem.py

   - O navegador abrirá automaticamente o app (http://localhost:8501)

3. No app:
   - Carrega o ficheiro CSV (usa exemplo.csv como teste)
   - Faz o download do ficheiro Excel gerado com os dados e contagens
